# Getting started

1. Enter the app directory

    ```bash
    cd pfp
    ```

2. Download the project dependencies using your favorite package manager:

    ```bash
    pnpm install
    ```

3. Run the frontend/backend servers:
    - **Automatically** (with tmux)

        ```bash
        bash run.sh
        ```

    - Manually

        First:

        ```bash
        pnpm run dev
        ```

        Then, in a separate terminal

        ```bash
        rapid run
        ```
