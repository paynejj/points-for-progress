# Getting started
1. First, download the project dependencies using your favorite package manager:
```bash
pnpm install
```

2. Next, Run the [Remix](https://remix.run) frontend:
```bash
pnpm run dev
```

3. Finally, run the Rapid rust backend:
```bash
rapid run
```
